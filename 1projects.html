Highlighted Projects

ToxiBERT

A full-stack AI-powered toxic comment classification web app using DistilBERT. Features include real-time classification across multiple toxicity labels (toxic, severe toxic, obscene, threat, insult, identity hate), voice input and audio output using Web Speech API, dynamic emoji-based sentiment visualization, comment history tracking, and a chatbot-style UI. Built using Flask, HTML/CSS/JS, Chart.js, and HuggingFace Transformers.

Medic - Brain Stroke Prediction

An AI-based health diagnostic system using Naive Bayes for early stroke prediction. The platform allows users to input health metrics such as BMI, hypertension, smoking habits, and heart disease. It provides risk insights and visual analysis to help medical professionals and users take proactive measures. Built using Python (scikit-learn), Flask, and interactive dashboards.

Smart Shopping Assistant Kiosk

A UX/UI prototyping project developed in Figma to enhance in-store grocery experiences. Based on user research and human-centered design principles, the kiosk guides customers to product locations, suggests smart shopping paths, and recommends deals based on past shopping behavior. Focuses on design thinking, usability, and real-world scenario mapping.

LearnMate

AI-powered educational platform using Gemini AI to generate personalized study materials including notes, quizzes, summaries, and topic explanations. Built as part of an academic collaboration, it dynamically adjusts content difficulty and structure based on learner progress. Frontend designed with HTML, CSS, JS while AI APIs were integrated using Node.js and Flask for content generation.
